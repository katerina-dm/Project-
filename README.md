Парсер государственных закупок
Система для автоматического поиска и сохранения закупок с официального портала zakupki.gov.ru.

Возможности
•	Поиск закупок по ключевым словам
•	Автосохранение в базу данных SQLite
•	 Веб-интерфейс для удобного просмотра
•	Статистика поисков и результатов

Структура проекта
Project-/
├── main.py              #  приложение
├── zakupki_parser.py    # Парсер закупок
├── db_manager.py        # Управление базой данных
├── requirements.txt     # Зависимости
├── data/
│   └── zakupki.db      # База данных SQLite
├── templates/          # HTML шаблоны
│   ├── index.html
└── static/
    └── style.css       # Стили

База данных
•	id - уникальный идентификатор
•	title - объект закупки
•	customer - заказчик
•	price - цена
•	link - ссылка на закупку
•	search_query - поисковый запрос
•	created_at - дата сохранения
•	
⚠Особенности работы
•	Для работы требуется Firefox браузер
•	Поиск выполняется в фоновом режиме (headless)
•	Время загрузки страницы - 5-10 секунд
•	Сохраняются первые 20 найденных закупок
•	Автоматическое обновление статистики

Основные компоненты:
•	Парсер - извлекает данные с сайта закупок
•	Веб-интерфейс - предоставляет удобный доступ
•	База данных - обеспечивает надежное хранение
•	Система экспорта - позволяет анализировать данные
